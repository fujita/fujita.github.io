FUJITA Tomonori

I've been playing in operating systems; mainly, I/O systems, file systems, and storage systems. You can find me in group pictures of Linux Kernel Developers Summit 2007, 2008, 2009, 2010, but who cares about what I look like?

Software

  • I founded and have maintained the RustyBGP project, a BGP implementation in the Rust Programming Language.

  • I founded and have maintained the GoBGP project, a BGP implementation in the Go Programming Language.

  • I founded and had maintained the Ryu SDN Framework project, Python libraries for networking.

  • I founded and had maintained the Swift3 project, adding the Amazon S3 API support to OpenStack Swift.

  • I founded and have maintained the target infrastructure in Linux SCSI subsystem with Mike Christie. It's known as scsi-target-utils or tgt.

  • I started Sheepdog project, a distributed storage system for KVM with MORITA Kazutaka and had worked until the first public release.

  • I founded and had maintained iSCSI Enterprise target software (IET), enables you to convert Linux machines into iSCSI storage systems. It's the most famous "iSCSI target", Google said on March 20, 2007.
